# SGBCC Seed Images

Drop SGBCC-specific images here to override the shared SHBCC defaults.

Any file with a matching filename will be used instead of the one in
`seed-content/images/`. Files not found here fall back to the SHBCC
originals automatically.

## Files you'll probably want to replace first

- `chamber-logo.png` — SGBCC logo
- `hero-image.png` — Homepage hero
- `board-president.png` — Sir Honksalot headshot
- `board-vice-president.png` — Marcus Delacroix headshot
- `board-treasurer.png` — Priya Narayan headshot
- `seal-testimonial.png` — Swan testimonial photo (it's not a seal anymore)
